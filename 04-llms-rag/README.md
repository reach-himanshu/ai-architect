# LLMs & RAG (Retrieval-Augmented Generation)

This module covers the core of the AI Architect role — understanding how Large Language Models work, how to build production RAG systems, and how to design reliable AI pipelines.

## 🚀 Roadmap & Sequence

### 1. Foundations of LLMs
*   **01. LLM Fundamentals:** Transformers, attention, tokenization, and the intuition behind how LLMs generate text.
*   **02. Prompt Engineering:** Few-shot, chain-of-thought, structured outputs, and system prompts.
*   **03. LLM APIs:** Working with OpenAI, Anthropic, and Gemini APIs — structured usage, retries, cost tracking.

### 2. Embeddings & Vector Search
*   **04. Embeddings:** What embeddings are, how similarity works, and choosing embedding models.
*   **05. Vector Databases:** Pinecone, Weaviate, Chroma, pgvector — indexing, querying, and trade-offs.

### 3. RAG Architecture
*   **06. RAG Fundamentals:** Chunking strategies, ingestion pipelines, retrieval patterns, reranking.
*   **07. Advanced RAG:** HyDE, query expansion, multi-hop retrieval, parent-child chunking.

### 4. Agents & Orchestration
*   **08. Agentic Patterns:** ReAct, tool use, multi-agent systems, memory architectures.
*   **09. LLM Evaluation:** RAGAS, hallucination detection, benchmarks, and building eval harnesses.

### 5. Production LLM Systems
*   **10. LLM Observability & Safety:** Tracing (LangSmith, Arize), cost/latency tracking, prompt injection defense, guardrails.

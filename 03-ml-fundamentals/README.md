# ML Fundamentals

This module builds the machine learning intuition required to make good AI architecture decisions. You don't need to implement ML algorithms from scratch — you need to understand trade-offs, failure modes, and when to use what.

## 🚀 Roadmap & Sequence

### 1. Core Concepts
*   **01. ML Fundamentals:** Supervised/unsupervised learning, train/val/test splits, overfitting, bias-variance.
*   **02. Embeddings & Representations:** What embeddings are, dimensionality, transfer learning, fine-tuning.

### 2. Evaluation & Selection
*   **03. Model Evaluation:** Metrics (precision/recall/F1/AUC), confusion matrices, evaluation pitfalls.
*   **04. Feature Engineering:** Preprocessing, encoding, normalization, and handling missing data.

### 3. Neural Networks (Intuition)
*   **05. Neural Network Intuition:** Forward pass, backprop intuition, activation functions, architecture trade-offs.
*   **06. Fine-Tuning & Transfer Learning:** When to fine-tune, LoRA/PEFT concepts, dataset requirements.

# Cloud Architecture for AI Systems

AI workloads are cloud-native. This module covers the cloud infrastructure decisions every AI architect must master.

## 🚀 Roadmap & Sequence

### 1. AI Services & Compute
*   **01. Cloud AI Services:** AWS vs GCP vs Azure AI offerings, managed vs self-hosted, vendor lock-in.
*   **02. GPU & Compute:** Instance types, on-demand vs spot, inference optimization.

### 2. MLOps on Cloud
*   **03. Managed ML Services:** SageMaker, Vertex AI, Azure ML — pipelines, endpoints, AutoML.
*   **04. Serverless for AI:** Lambda/Cloud Run for inference, cold starts, cost models.

### 3. Data & Cost
*   **05. Storage for ML:** S3/GCS, data lakes, Parquet, feature stores, versioning.
*   **06. Cost Optimization:** Reserved instances, spot, prompt caching, rightsizing, FinOps.

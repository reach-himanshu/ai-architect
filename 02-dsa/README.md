# Data Structures and Algorithms (DSA) in Python

This module provides a structured approach to learning DSA using Python. Each topic consists of a **Theory Guide** (`.md`) and a **Practical Notebook** (`.ipynb`) containing implementations and common interview problems.

## 🚀 Roadmap & Sequence

### 1. Foundations
*   **01. Big O Notation:** Time and Space complexity.

### 2. Linear Data Structures
*   **02. Arrays & Strings:** Contiguous memory, slicing, and common string manipulations.
*   **03. Linked Lists:** Singly and Doubly linked lists.
*   **04. Stacks:** Last-In-First-Out (LIFO) logic and applications.
*   **05. Queues & Buffers:** First-In-First-Out (FIFO) and Circular Buffers.
*   **06. Hash Tables:** Dictionaries, collision handling, and efficiency.

### 3. Non-Linear Data Structures
*   **07. Recursion:** Understanding the call stack and base cases.
*   **08. Trees:** Binary Trees, Binary Search Trees (BST), and Heaps.
*   **09. Graphs:** Representations (Adjacency Matrix/List) and Traversals (BFS/DFS).

### 4. Essential Algorithms
*   **10. Searching:** Linear vs. Binary Search.
*   **11. Sorting:** Bubble, Insertion, Merge, and Quick Sort.
*   **12. Two Pointers & Sliding Window:** Optimized techniques for arrays.

### 5. Advanced Paradigms
*   **13. Greedy Algorithms:** Local optimization to find global solutions.
*   **14. Dynamic Programming (DP):** Overlapping subproblems and memoization.
